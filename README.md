# Advanced AI Translator

An AI-powered translator built using Python and Streamlit.

## Features
- Translate text into multiple languages
- Automatic language detection
- Dark/Light mode
- Image text extraction (OCR)
- Voice output using gTTS
- Translation history
- Download translated text

## Technologies Used
- Python
- Streamlit
- Googletrans
- gTTS
- Pillow
- Pytesseract

## Run Locally

pip install -r requirements.txt

streamlit run app.py

Note: Install Tesseract OCR separately and update the Tesseract path in app.py if needed.
